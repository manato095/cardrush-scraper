# CardRush Scraper

This is a simple Python script to scrape product names and prices from the CardRush Pok√©mon card site:  
https://www.cardrush-pokemon.jp/phone/

## Requirements

Install dependencies with:

```
pip install -r requirements.txt
```

## Usage

Run the scraper:

```
python scraper.py
```

The results will be saved in `output/cardrush_products.xlsx`.
